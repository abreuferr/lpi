# NETSTAT
#
# aplicativo utilizado para exibir informacoes de conexao, tabela
# de rotas, estatistica das interfaces de rede, conexao do tipo
# masquerading e membros da conexao multicast
#
# netcast [opcao] [protocolo]
#
# -ie - semelhante a opcao "ifconfig" ou "ip addr"
# -t - protocolo tcp
# -u - protocolo udp
# -n - nao resolve o nome
# -a - todas as conexoes
#
# -tuna - as principais opcoes a serem utilizadas

# /ETC/HOSTS
#
# o arquivo /ETC/HOSTS pode ser configurado para resolucao
# de nomes no host local. Desta forma, nao eh necessario
# a implementacao de um servidor de DNS.
#
# [IP]          [HOST]  [DOMINIO]       [HOST]
# 192.168.0.1   gateway.dominio.local   gateway

# /ETC/RESOLV.CONF
#
# o arquivo indica quais servidores de DNS a estacao
# deve utilizar para resolver o nome/DNS
#
# [NAMESERVER]  [IP]
# [SEARCH]      [DOMINIO]
# [DOMAIN]      [DOMINIO]
#
# nameserver    192.168.0.5
# nameserver    192.168.0.6
# search        dominio.local
# domain        dominio.local
