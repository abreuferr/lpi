######
# INSTALCAO VIA CODIGO FONTE
######
#
# procedimentos para a instalacao de um programa atraves
# do codigo fonte. um dos motivos para fazer este tipo de
# instalacoes eh que normalmente o software eh lancado
# primerio via codigo fonte em vez de empacotar.
#

# DIRETORIO
#
/usr/src/

# DOCUMENTACAO
#
README
INSTALL
COPYING

# DESCOMPACTACAO
#
$ tar xzvf arquivo.tar.gz
$ tar xjvf arquivo.tar.bz2
$ tar xJvf arquivo.tar.xz

# CONFIGURACAO
#
# o comando "configure" tem por objetivo o de verificar as dependencia, 
# como por exemplo, biblioteca e compilador.
# 
# criacao do aqruivo Mikefile ou Makefile.ini
#
# --prefix=[DIR] ; informa qual diretorio do sistema de arquivos o 
# o binario, os manuais e afins deverao ser instalados.
#
$ ./configure --prefix=/usr/src

# COMPILACAO
#
$ make ; comando utilizado para compilar o programa

# INSTALACAO
#
$ make install ; comando utilizado para instalar os componentes do programa

# LIMPEZA
#
$ make clean ; comando utilizado para apagar os arquivos objetos

# DESINSTALACAO
#
$ make uninstall ; comando utilizado para desinstalar o programa
